Welcome to my fibonacci visualizer page!

The idea of the algorithm is as follows

  1. Check to see if the number(n) is 1 or 2, if so return 1
  2. Create a list and add 1 twice to the list
  3. Repeat the following n - 2 times, since we have already dealt with the case of the first two fibonacci numbers
    1. add the previous fibonacci number and the previous of the previous fibonacci number together and then add the result to the list
  4. Return the last thing added to our list and that's your nth fibonacci number! :)
