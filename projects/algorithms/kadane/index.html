Kadane's Algorithm

  • Purpose: Solves the contiguous array with largest sum problem
  • Founder: Jay Kadane
  • Run Time: O(n)

Formal Definiton

Input: S[1...n] where ∀ i, Si ∈ ℝ & n ∈ ℝ

Output: SS[1...m] where m ∈ ℝ & ≤ n. ∀ i, SSi ∈ ℝ and SS is the max contiguous array of the largest summation

Psuedocode

						
Kadane(S):
	current_max = overall_max = S[0]							// set the maxes

	current_contiguous = []
	contiguous = []									// contiguous array of maxes
	current_contiguous.push(S[0])

	for i from 1 to n:								// iterate through all values
		current_max = max(current_max, S[i] + current_max)

		// check where current_max came from
		if current_max came from S[i] + current_max:
			current_continguos.push(S[i])
		else:
			current_contiguous = [current_max]										

		overall_max = max(current_max, overall_max)

		// check to see if we have a new current max
		if overall_max came from current_max:
			contiguous = current_contiguous

	return contiguous, overall_max
