0-1 Knapsack Algorithm

How to use this program

  • First, enter a list of weights
  • Second, enter a list of values, corresponding to those weights
  • Third, enter a number which will represent the capacity of a bag to hold the items
  • Finally, either run through the algorithm by clicking the play button or step through the algorithm with the forward and backward buttons

The facts

  • Purpose: maximize value that can be attained with certain restrictions
  • Space Complexity: O(nW)
  • Time Complexity: O(nW)

Formal Definiton

Input: W[1...n] where W is the weights of the items, V[1...n] where V is the value of each item, C which represents the max capacity of the sack. Wi, Vi, C, n ∈ ℝ

Output: MAX, where MAX is the most value that can be placed in the knapsack & MAX ∈ ℝ, MAX ≥ 0

Psuedocode

Knapsack(W[1...n], V[1...n], C):
	Create a new nxn array called knap  	

	for i from 0 to C:
		set knap[0][i] to 0

	for i from 1 to n:
		for j from 0 to C:	

			// check to see if the weight is too large
			if W[i] > j:
				knap[i][j] = knap[i - 1][j]

			// take the maximum of the previous item and add the value of the current item to it
			else:
				knap[i][j] = max(knap[i - 1][j], V[i] +  knap[i - 1][j - W[i]])

	return knap[n][C]
