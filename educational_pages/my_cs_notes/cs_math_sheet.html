Exponents

  • \(x^a * x^b = x^{a + b}\)
  • \(\frac{x^a}{x^b} = x^{a - b}\)
  • \((x^a)^b = x^{ab}\)
  • \(x^n + x^n = 2x^n\)
  • \(2^n + 2^n = 2^{n + 1}\)

Logarithms

  • \(x^a = b \Leftrightarrow \lg_xb = a\)
  • \(\lg_aB = \frac{\lg_cB}{\lg_cA} \hspace{0.25cm}where\hspace{0.25cm} A,B,c \in \mathbb{Z^+}\)

    • Proof:

    • \(x = \lg_cB,\hspace{0.25cm} y = \lg_cA,\hspace{0.25cm} z = \lg_aB\)
    • \(c^x = b,\hspace{0.25cm} c^y = a,\hspace{0.25cm} a^z = b\)
    • \(c^x = b = a^z = (c^y)^z = c^{yz}\)
    • \(\rightarrow x = yz \rightarrow z = \frac{x}{y} = \frac{\lg_caA}{lg_cB}\)

  • \( \lg_nAB = \lg_nA + \lg_nB \)

    • Proof:

    • \(x = \lg_nA,\hspace{0.25cm} y = \lg_nB,\hspace{0.25cm}z = \lg_nAB\)
    • \(n^x = A,\hspace{0.25cm}n^y = B,\hspace{0.25cm}n^z = AB\)
    • \(AB = n^xn^y = n^{x + y} \rightarrow z = x + y = \lg_nA + \lg_nB\)

  • \(\lg_n(\frac{A}{B}) = \lg_nA - \lg_nB)\)
  • \(\lg_n(A^B) = B\lg_nA\)
  • \(\lg_nX < X\hspace{0.25cm}\bigvee X > 0\)
  • \(\lg_n1 = 0\)

Series

  • $$\sum_{i=0}^{N}2^i = 2^{N+1}-1$$
  • $$\sum_{i=0}^{N}a^i = \frac{a^{N+1} - 1}{a-1}$$
  • $$\sum_{i=0}^{N}a^i = \frac{1}{1-a},\hspace{0.25cm}0 < a < 1$$
  • $$\sum_{i=1}^{N}i = \frac{N*(N + 1)}{2}$$
  • $$\sum_{i=1}^{N}i^2 = \frac{N*(N+1)*(2N+1)}{6}$$
  • $$\sum_{i=1}^{N}i^k = \frac{N^{k+1}}{|k+1|}$$

Modular Arithmetic

A is congruent to B modulo N {A ≡ B % N} if N divides A - B

If N is a prime number the following must follow:

  1. AB ≡ 0 % N ⇔ A ≡ 0 % N or B ≡ 0 % N
    • If N divides a product of two numbers, it divides at least one of the two numbers
  2. AX ≡ 1 % N has a unique solution ∀a in range(0 : N)
  3. x2 ≡ A mod N has either two solutions ∀a in range(0 : N) or no solutions
