Graphs

What is a graph? A graph is a set of vertices all linked together by a set of edges. Graphs are not very difficult to find in the real world. We find them when we network with people to see who we know, routing between cities to see how long it takes to get from one city to another, tournament brackets to see what teams play what and several other areas in life.

Common graphs

  • Simple: Each edge connects two different vertices and no two edges connect the same pair of vertices.
  • Multigraph: A graph in which there can be multiple edges connecting the same vertices.
  • Pseudograph: Graph that can include loop and multiple edges to other vertices or to a single vertex.
  • Undirected: Graph in which the each edge doesn't care as to what the start or endpoint vertices.
  • Directed: Each edge is associated with an ordered pair of vertices. Such that Edge(u,v) \(\neq\) Edge(v,u).
  • Simple directed: Directed graph with no loops nor multiple edges.
  • Directed multigraph: Directed graph that contains multiple directed edges.
  • Mixed: Graph which contains directed and undirected edges.

Terminology

  • Loop: A vertex which has an edge to itself.
